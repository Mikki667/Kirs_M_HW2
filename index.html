Counter Documentation

Reusable Counter Component

Overview

This project demonstrates a reusable counter component built using JavaScript OOP. The counter keeps track of its own state and updates the interface when users click the buttons.

How To Use

Create a container element in HTML and instantiate the Counter or StepCounter class in JavaScript. The component accepts an initial value and optionally a step value for custom increments.
